Cada diagrama agora segue o padrão internacional BPMN 2.0 com estilo Bizagi:
- ✅ **Tarefas**: Retângulos brancos com bordas cinzas (`#90A4AE`)
- ✅ **Gateways**: Losangos amarelos (`#FDD835`) para decisões
- ✅ **Eventos de Início**: Círculos verdes (`#43A047`)
- ✅ **Eventos de Fim**: Círculos vermelhos com borda grossa (`#B71C1C`)
- ✅ **Setas de Fluxo**: Cinzas (`#546E7A`) para fluxo normal, vermelhas para exceções

### 5. **Detalhes Técnicos dos Diagramas**

#### Produção Industrial (Linha de Envase)
- 4 tarefas principais: Receber MP → Inspeção → Produção/Envase → Controle Qualidade
- 1 gateway exclusivo (MP Conforme?)
- Swimlanes implícitas em layout horizontal limpo

#### Manutenção Industrial
- Fluxo de Ordens de Serviço (Corretiva e Preventiva)
- Gateway para tipo de manutenção
- Branch para aprovação/rejeição

#### Pronto Socorro (Triagem de Manchester)
- 3 tarefas: Admissão → Triagem → Atendimento por Prioridade
- Gateway de prioridade clínica
- Cores específicas para urgência (azul, verde, amarelo, vermelho)

#### Logística de Distribuição
- Gestão completa de pedidos
- Verificação de stock
- Separação e expedição

#### Onboarding RH
- Processo de integração de colaboradores
- Documentação → Acessos → Integração
- Gateway para conclusão
